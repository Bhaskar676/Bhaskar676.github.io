Gaming Post Automation Agent – Privacy Policy

Effective Date: 1 Aug 2025

Introduction

This Privacy Policy explains how the Gaming Post Automation Agent (the “Agent”) collects and uses information. The Agent is a local automation script that fetches gaming-related news articles, uses generative AI to create captions and images, and publishes posts on a connected Instagram professional account. The Agent is operated solely by you for your own Instagram account and is not provided to other users.

Meta requires developers to provide a publicly accessible privacy policy detailing what information is collected, how it is used, and how users can request deletion. This policy fulfils that requirement for the Agent.

Information Collected

  • News content: The Agent retrieves news articles using a third‑party news API (e.g., GNews). The script processes the article title, description and URL to generate captions and does not store the full article text.
  • AI‑generated content: The Agent calls OpenAI’s API to generate captions and images. It stores the generated caption and downloads the resulting image file locally.
  • Instagram account details: During setup, you provide a long‑lived Instagram User Access Token and the Instagram Account ID. The Agent uses these credentials to create media containers and publish posts via the Instagram Graph API.

The Agent does not collect personal data from other Instagram users. It only interacts with your own professional account using credentials you supply. The script does not collect names, email addresses or any sensitive personal information from end users.

How Information Is Used

  1. Content creation: News article metadata is summarised to draft an engaging caption using OpenAI’s language model. An image prompt is generated and sent to OpenAI’s image generation API to create a relevant illustration. The generated caption and image are then prepared for posting.
  2. Publishing posts: The Agent uses the Instagram Graph API to create a media container and publish the generated post to your Instagram professional account.

The Agent does not share your credentials, captions or images with any third party. All API keys and tokens remain on your local machine and are used only to perform the tasks described.

Data Retention

The Agent stores generated captions and downloaded images locally until you choose to delete them. It does not persistently store news articles or personal data. You may delete the generated files at any time.

Data Deletion

Because the Agent operates locally and only uses data you provide, there is no external database to purge. If you wish to remove generated content or revoke access:

  • Delete any caption or image files stored on your computer.
  • Remove your API keys and tokens from the script.
  • Revoke the Instagram User Access Token via the Meta app dashboard or by using the token revocation tools in Business Suite.

Third‑Party Services

The Agent relies on the following services:

  • GNews (or similar news API): Provides current gaming news articles. Review the service’s terms and privacy policy before use.
  • OpenAI API: Generates captions and images. Use of the API is subject to OpenAI’s Terms of Service and Privacy Policy. Only prompts and generated content are sent to OpenAI; no personal user data is transmitted.
  • Meta Instagram Graph API: Publishes posts to your Instagram professional account. Data transmitted includes your access token, image URL and caption.

Security

You are responsible for securing the API keys and access tokens used by the Agent. Store these credentials in a secure manner and restrict access to the machine running the script. Do not share your tokens publicly, as they provide access to your Instagram account and APIs.

Changes to This Policy

As the operator of the Agent, you may update this policy to reflect changes in functionality or applicable laws. Any updates should continue to explain what information is collected, how it is used and how it can be deleted, in accordance with Meta’s privacy policy expectations.

Contact

If you have questions about this Privacy Policy or wish to request deletion of information associated with the Agent, please contact the operator of the Agent (i.e., you) via the email address associated with your Meta Developer account.
